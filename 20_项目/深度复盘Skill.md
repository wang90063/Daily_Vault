# 深度复盘Skill

## Context (目标与背景)

为 OrbitOS 设计并实现一个"深度复盘"Skill，帮助用户记录每天做了什么、哪些做得好、哪些需要改进，覆盖工作和生活两个维度。

**目标:** 建立一套可持续运行的每日复盘工作流，明确输入、处理流程和输出格式。

**成功指标:**
- [x] 完成方法论调研（4 个参考来源，详见 [[深度复盘skill项目]]）
- [ ] 确定复盘工作流设计（输入项、处理逻辑、输出格式）
- [ ] 实现 `/review` skill 并可正常使用

**限制条件:**
- 依赖项: 先完成方法论调研，再设计工作流

---

## Actions (计划与步骤)

### 阶段1: 方法论调研

- [x] 收集 4 个复盘方法参考（Atlassian / KPT / ORID / AAR）
- [x] 整理各方法的核心框架和适用场景（详见 [[深度复盘skill项目]]）
