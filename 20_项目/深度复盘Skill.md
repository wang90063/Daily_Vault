# 深度复盘Skill

## Context (目标与背景)

为 OrbitOS 设计并实现一个"深度复盘"Skill，帮助用户记录每天做了什么、哪些做得好、哪些需要改进，覆盖工作和生活两个维度。

**目标:** 建立一套可持续运行的每日复盘工作流，明确输入、处理流程和输出格式。

**成功指标:**
- [ ] 完成方法论调研（3-5 个参考来源）
- [ ] 确定复盘工作流设计（输入项、处理逻辑、输出格式）
- [ ] 实现 `/review` skill 并可正常使用

**限制条件:**
- 依赖项: 先完成方法论调研，再设计工作流

---

## Actions (计划与步骤)

### 阶段1: 方法论调研

- [ ] 收集 3-5 个复盘方法参考（视频/博客）
- [ ] 整理各方法的核心框架和适用场景
- [ ] 选定最适合日常使用的复盘模型

### 阶段2: 工作流设计

- [ ] 定义输入项（用户需要提供什么）
- [ ] 设计处理流程（AI 如何帮助整理和分析）
- [ ] 确定输出格式（复盘笔记的结构）

### 阶段3: Skill 实现

- [ ] 编写 skill prompt 和工作流脚本
- [ ] 创建复盘笔记模板
- [ ] 测试并迭代优化
