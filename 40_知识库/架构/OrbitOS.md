# OrbitOS

OrbitOS 是一种基于 **Obsidian** 和 **AI Agents** 构建的个人操作系统框架。其核心理念是“万物绕我运行”（Everything orbits around the user），通过结构化的目录和自动化脚本实现知识与任务的无缝管理。

## 核心原则
1. **动量守恒**: 鼓励信息处于运动状态（从收件箱向项目或知识库移动），防止知识停滞。
2. **扁平化与连接**: 弱化文件夹层级，强调通过 Wikilinks 建立神经元般的连接。
3. **人机协作**: 利用 Agent Skills 自动化重复性劳动（如早间规划、资讯聚合）。

## 目录规范 (The Numbering System)
- `00-09`: 捕捉 (Capture)
- `10-19`: 时间 (Time/Journal)
- `20-29`: 执行 (Execution/Projects)
- `30-49`: 沉淀 (Archive/Knowledge)
- `50-89`: 外部资源 (Resources)
- `90-99`: 系统底层 (System)
