# GRAI复盘模型

GRAI 代表 Goal / Result / Analysis / Insight，是一个适用于季度、项目或重大事件的大复盘流程。它先审视当时设定的目标是否符合 SMART，再用目标与结果的差距驱动分析，期间鼓励列举大量原因并通过连续的“五个为什么”拆解，最终提炼出可复用而不显而易见的洞察，避免直接落入批判或责备。

Goal 帮助确认目标设定是否合理；Result 让差距可视化；Analysis 强调无责地列出所有可能因素，结合 [[复盘问题梯度]] 的解释层；Insight 则是反常识或模式化的结论，有助于指导后续行动或知识沉淀。
